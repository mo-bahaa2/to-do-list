# إنجازاتك يا بطل (To-Do List) 🚀

تطبيق ويب ذكي لتنظيم المهام بتصميم (Hand-drawn) مرح باللغة العربية.

## المميزات:
- تصميم عفوي ورسم يدوي (Hand-Drawn UI).
- اختيار أوقات المهمة بضغطة زر.
- احتفال تفاعلي (Confetti 🎉) عند إنجاز المهام.
- أصوات كوميدية وتحفيزية عند الإنجاز.
- واجهة مستخدم (UX) مصرية ممتعة.
- متجاوب مع جميع الشاشات.

## التقنيات المستخدمة:
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)
